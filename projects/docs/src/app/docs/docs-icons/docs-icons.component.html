Example

To use import the icon component and the icons you would like to use. By explicitly importing the icons you want to use your application will only bundle the icons necessary improving performance. (Demo icons provided by Clarity Design System and Font Awesome)



Icon Collections

You can also add specific collections of icons or all the icons provided in one statement. While convenient it is not recommended for performance as your application will load icons it may not be using.

Custom Icons

Using the same API to select icons you can add your own custom SVG icons. If you provide an icon with the same existing name as another icon in the registry it will be overridden.
