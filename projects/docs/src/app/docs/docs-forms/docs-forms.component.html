Forms

The li-form element provides basic styles to a form group of other lithium based inputs. Lithium inputs apply styles and provide automatic utilities for accessibility (a11y). Lithium inputs work with your framework or form library of choice works out of box. Lithium does not provide a way to layout components. You can use any grid system with Lithium, we recommend Blueprint CSS.
